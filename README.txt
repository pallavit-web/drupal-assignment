# Drupal Assignemnt

## Introduction
This module provides a Ajax form , 2 web forms with validation 
Ajax Form - http://localhost/drupal-assignment/?q=customajax_forms
Web form1- Feedback form - http://localhost/drupal-assignment/?q=node/19
Web form2- Contact Form - http://localhost/drupal-assignment/?q=contact-form

## Requirements
- Drupal 7.79 orabove
- Entity API module (7.x-1.x)
- views module
- webforms module


## Installation
1. git clone https://github.com/pallavit-web/drupal-assignment
2. Download the module and place it in `sites/all/modules/
 -chmod -R 775 sites/default/files
  Go to Admin > Modules and enable the below module.
 -Custom AJAX Forms
 -Diwali Gifts Module

## Database
